# Kickoff

## Goal
Read the product idea and prepare the operational foundation.

## Steps
1. Read `.braddock/memory/vision.md`
2. Verify that the files below exist, creating any that are missing:
   - .braddock/memory/prd.md
   - .braddock/memory/architecture.md
   - .braddock/memory/spec.md
   - .braddock/memory/tasks.json
   - .braddock/memory/decisions.md
   - .braddock/memory/status.md
   - .braddock/board/kanban.json
3. Record in `.braddock/memory/status.md` that the project has been initialized
4. Suggest the next skill to run: `/create-prd`
