# Create PRD

Use the `pm` agent.

## Goal
Transform `.braddock/memory/vision.md` into a clear PRD.

## Required output in `.braddock/memory/prd.md`
- problem
- context
- product goal
- target audience
- scope
- out of scope
- success metrics
- risks
- open questions

When done, update `.braddock/memory/status.md` and indicate `/create-spec`.
