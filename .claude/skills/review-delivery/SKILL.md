# Review Delivery

Use the `qa` agent with support from `tech-lead`.

## Goal
Verify adherence of what was delivered.

## Validate
- acceptance criteria
- risks
- regressions
- adherence to spec
- adherence to PRD

## Outputs
Update `.braddock/memory/status.md` with:
- what was validated
- gaps found
- pending items
- final recommendation
